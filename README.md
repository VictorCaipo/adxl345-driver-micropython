# ADXL345 driver on Mycropython

1) At your glance:
- Here I have tried to develop a driver for the sensor called ADXL345. You can set up both I2C and SPI bus serial communication, here we are using I2C bus. If something goes wrong, let me know.

2) Requirements:
- On the folder called "01.documentation_resources" I have attached two main documents.
    - the specific official version of the microcontroller provided by ESPRESSIF
    - the official adxl345 datasheet provided by Analog Devices 
- Tools:
    - ESP32 devkitc v4 (microcontroller)
    - ADXL345 (sensor/accelerometer)
    - I2C (serial_bus_communication)
